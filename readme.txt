== Description ==

**BanglaPress WooCommerce Payment:** bKash,Nagad,Rocket,Upay and Partial Payment streamlines mobile payments for Bangladeshi businesses with gateways like **bKash**, **Nagad**, **Rocket**, and **Upay**. It supports one-click payments, **partial payments**, and customizable checkout, along with new features like a **Buy Now Button**, **Dynamic Shipping Zones**, compatibility with **Simple and Variable Products**, a Custom CSS Design, simplified **Payment ID** entry, and downloadable **Order Details** as images, ensuring a seamless shopping experience.

### New Features

-**Buy Now Button:** This button is added after the "Add to Cart" button for quick purchases.
-**Dynamic Shipping Zone:** Supports Flat Rate and Free Shipping based on selected zones.
-**Types of Products:** Compatible with both Simple Products and Variable Products.
-**Custom Design:** Includes a unique CSS design for the "Buy Now" button.
-**Payment ID:** Requires only the last 4 digits of the payment number, eliminating the need to enter a full bKash transaction ID.
-**Download Order Details:** Customers can now download their order details as an image after completing payment.

== Installation ==

1. Download the plugin and upload it to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your settings under **WooCommerce > Settings > BanglaPress Payment** to set up mobile payment options, customize labels, and more.


== Usage ==

**Fix Checkout Page Issues**
If the checkout page is unresponsive, try switching to the Block Editor or Classic Editor,first install [Classic Editor](https://wordpress.org/plugins/classic-editor/) 
removing the default checkout, and adding the shortcode `[woocommerce_checkout]`

To add a payment button to any page, use the following shortcode format:

`[bkash_payment_button product_id="123"]`

- `product_id` - Specify the WooCommerce product ID.
- `label` - (Optional) Customize the button label.
  
**Example**:  
`[bkash_payment_button product_id="101" label="Quick Pay with bKash"]`

== Frequently Asked Questions ==

= How do I add a payment button to a page? =  
Use the shortcode `[bkash_payment_button product_id="123"]` and place it on any page or post. Replace `"123"` with the actual WooCommerce product ID.

= Can I change the button color and style? =  
Yes! You can customize the button style directly in the shortcode, for example:  
`[bkash_payment_button product_id="101" label="Pay Now" style="background-color: #E2126E; color: white;"]`

= Can I control which payment gateways are available? =  
Yes, you can enable or disable specific gateways from the settings under WooCommerce > BanglaPress Payment.

= Does this plugin support partial payments? =  
Absolutely. Partial payments can be set up so customers pay only a portion initially and the remainder upon delivery.
